🔹 Lệnh tải file/code mới từ GitHub

Mở CMD/Terminal trong thư mục dự án của bạn (nơi đã clone).
Ví dụ:

cd C:\xampp\htdocs\HeThongTuyenDung


Chạy lệnh:

git pull origin main


(hoặc git pull origin master nếu repo dùng nhánh master)
-------------------------------------------------------------------------------------

# Thêm tất cả file mới vào git
git add .

# Commit lại (ghi chú thay đổi)
git commit -m "Thêm file mới"

# Đẩy lên GitHub
git push origin main

-------------------------------------------------------------------------------------

🔹 Các bước up file cụ thể bằng Git

Mở CMD/Terminal tại thư mục dự án.

Thêm file cần upload, ví dụ bạn muốn up index.php:

git add index.php


Commit lại (ghi chú thay đổi):

git commit -m "Thêm file index.php"


Push lên GitHub:

git push origin main


(nếu repo dùng nhánh master thì thay main thành master)

🔹 Nếu muốn up nhiều file cụ thể

Ví dụ bạn muốn up index.php và style.css thì:

git add index.php style.css
git commit -m "Thêm index.php và style.css"
git push origin main

--------------------------------------------------------------------------------------

🔹 Trường hợp 1: Chỉ chỉnh sửa file (chưa git add)

Nếu bạn chỉ sửa file nhưng chưa git add, thì dùng:

git checkout -- .


👉 Toàn bộ file sẽ quay về trạng thái như lần cuối commit/pull.

🔹 Trường hợp 2: Đã git add nhưng chưa commit

Nếu bạn đã git add rồi nhưng chưa commit:

git reset --hard


👉 Xóa staging (vùng git add) và hủy hết thay đổi file.

🔹 Trường hợp 3: Đã commit nhưng chưa push

Nếu bạn đã lỡ commit nhưng chưa push, và muốn quay về giống GitHub:

git reset --hard origin/main


(hoặc origin/master nếu repo dùng master)
👉 Toàn bộ code trên máy sẽ quay về giống y hệt repo trên GitHub.

⚠️ Lưu ý: Các lệnh này sẽ mất hết thay đổi bạn vừa làm, không thể khôi phục lại nếu chưa backup.
